# Insurance Fraud Detection Using Machine Learning

## Activity 1.1 – Specify the Business Problem

Insurance provides financial protection against losses and damages. However, in recent years, fraudulent insurance claims have increased significantly. Fraudulent claims occur when individuals attempt to obtain benefits they are not entitled to by providing false or misleading information.

These fraudulent claims cause financial losses to insurance companies and increase operational costs. The number of fraud cases detected is much lower than the actual number of fraudulent activities committed.

Therefore, there is a need to develop an intelligent system that can automatically detect whether an insurance claim is fraud or legitimate based on various claim-related parameters.

The objective of this project is to build a Machine Learning model that classifies insurance claims as:

- Fraud Claim  
- Legitimate Claim  

This system will help insurance companies reduce financial losses and improve claim processing efficiency.
